Constructor

The constructor property returns the function that created the Array prototype.
For JavaScript arrays the constructor property returns:
function Array() { [native code] }

For JavaScript objects the constructor property returns:
function Object() { [native code] }
contoh 

                
                    const listBuah = ['apel', 'ceri', 'duku', 'jambu', 'mangga'];
                    const test = listBuah.constructor;
                    document.getElementById('resultconstructor').innerHTML = test;

hasilnya: ada di dalam div dengan id resultconstructor

CopyWithin

copyWithin() adalah salah satu method JavaScript yang dapat kalian gunakan untuk menyalin elemen array ke posisi lain dalam array, dan menimpa nilai yang ada. Metode ini tidak menambahkan lebih banyak item ke array

array.copyWithin(target, start, end);

                    const hasilCopyWithin = buah.copyWithin(2, 0);
                    document.getElementById('resultcopywithin').innerHTML = hasilCopyWithin;

hasilnya: ada di dalam div dengan id resultcopywithin

Entries

The entries() method returns an Array Iterator object with key/value pairs.
For each item in the original array, the new iteration object will contain an array with the index as the key, and the item value as the value:

[0, "Banana"]
[1, "Orange"]
[2, "Apple"]
[3, "Mango"]

entries() does not change the original array.

Contoh:

                
                    const buah = ['apel', 'bangkoang', 'ceri', 'duku', 'jambu', 'kedongdong', 'mangga', 'nanas' ];
                    const hasilentries = buah.entries();
  
                    for (let hasilAkhir of hasilentries) {
                      document.getElementById("resultentries").innerHTML += hasilAkhir + "<br>";
                    }

hasilnya: ada di dalam div dengan id resultentries

Every

The every() method returns true if all elements in an array pass a test (provided as a function).

The method executes the function once for each element present in the array:

  • If it finds an array element where the function returns a false value, every() returns false (and does not check the remaining values)
  • If no false occur, every() returns true

every() does not execute the function for empty array elements.

every() does not change the original array

every() adalah salah satu method array JavaScript yang dapat kalian gunakan untuk memeriksa apakah semua elemen dalam array lulus tes yang disediakan didalam fungsi. Metode tersebut mengeksekusi fungsi satu kali untuk setiap elemen yang ada di dalam array. Jika menemukan elemen array di mana fungsi mengembalikan nilai false, maka method tersebut akan mengembalikan false dan tidak memeriksa nilai yang tersisa.

Begitupun sebaliknya, jika tidak ada yang salah, maka method tersebut akan mengembalikan true. Dan dari method tersebut tidak merubah array asli

Contoh:

                
                    const bilangan = [3, 5, 7, 9, 20] ;
                    const usia = [30, 34, 38, 40, 54];

                    const cekBilangan = (angka) => angka > 2 //Cek apakah angka diatas 2
                    const cekUsia = (umur) => umur > 32 ; //Cek apakah usia diatas 32
                    
                    const pengecekanBilangan = bilangan.every(cekBilangan); //Cek apakah semua bilangan diatas 2
                    const pengecekanUsia = usia.every(cekUsia); //Cek apakah semua usia diatas 32

                    const hasilEvery = `Apakah semua bilangan diatas 2 ? : ${pengecekanBilangan}, dan apakah semua usia diatas 32 ? :  ${pengecekanUsia}`;
                    document.getElementById('resultevery').innerHTML = hasilEvery;

hasilnya: ada di dalam div dengan id resultevery

Fill

The fill() method fills specified elements in an array with a static value.

You can specify the position of where to start and end the filling. If not specified, all elements will be filled.

fill() overwrites the original array.

Contoh:

                
                    const buah = ['apel', 'bangkoang', 'ceri', 'duku', 'jambu', 'kedongdong', 'mangga', 'nanas' ];
                    const hasilFill = buah.fill("Kiwi",2,4);
                    document.getElementById('resultfill').innerHTML = hasilFill;

hasilnya: ada di dalam div dengan id resultfill

Filter

Metode larik filter() membuat larik baru bersama elemen yang termasuk dalam kriteria yang ditentukan dari larik yang ada:

Contoh:

                        return bilangan.filter(angka => angka > 4);
                    }
                    document.getElementById('resultfilter').innerHTML = contohArrFilter();

hasilnya: ada di dalam div dengan id resultfilter

Find

Metode find() :

Contoh:

Foreach

Metode foreach() kegunaannya untuk melakukan iterasi pada array Javascript

Contoh:

                
                    const contohforeach = function() {
                        const buah = ['apel', 'bangkoang', 'ceri', 'duku', 'jambu', 'kedongdong', 'mangga', 'nanas' ];
                        let elemenli = ''
                        const buatElement = buah.forEach((fruit, index) => {
                            elemenli += `<li>index ke:${index}: ${fruit}</li>`
                        });
                        return `<ol>${elemenli}</ol>`;
                    }
                    document.getElementById('resultforeach').innerHTML = contohforeach();

hasilnya: ada di dalam div dengan id resultforeach

Includes

Metode includes() kegunaannya untuk mengecek array, apakah didalamnya ada suatu value, nilai kembaliannya adalah true atau false

Contoh:

                
                    const contohIncludes = function() {
                        const buah = ['apel', 'bangkoang', 'ceri', 'duku', 'jambu', 'kedongdong', 'mangga', 'nanas' ];
                        const namaBuah = 'ceri';
                        const cekBuah = buah.includes(namaBuah); // --> cek apakah ada ceri termasuk dalam array buah ?
                        return `Ternyata ${namaBuah} ${cekBuah ? 'ada' : 'tak ada'} dalam array`
                    }
                    document.getElementById('resultincludes').innerHTML = contohIncludes();

hasilnya: ada di dalam div dengan id resultincludes
