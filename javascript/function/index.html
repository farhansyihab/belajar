Belajar Function

Function Declaration

Deklarasinya seperti kebanyakan pembuatan function pada setiap bahasa pemograman, Contoh:

                
                    function tampilPesan(nama) {
                        console.log(`Halo ${nama}`)
                    }
                    
                    tampilPesan('Farhan')

Function Expression

cara pembuatan function ini adalah dengan mendeklarasikan variable, lalu variable itu diisi dengan function

cara pemanggilan function ini sama dengan cara pemanggilan Function Declaration, Contoh:

                
                    const sampaikanSalam = function(nama, waktu) {
                        console.log(`Halo ${nama}, selamat ${waktu}`)
                    }
                    sampaikanSalam('Farhan', 'pagi')

Arrow Function

Arrow function adalah expression function yang lebih modern (ini adalah pendapat pribadi farhan)

Bisa juga dibilang ini adalah expression function yang lebih ringkas

cara pemanggilan function ini sama dengan cara pemanggilan Function Declaration, Contoh:

                
                    const ucapkanSalam = (nama, waktu) => console.log(`Halo ${nama}, selamat ${waktu}`) ;
                    ucapkanSalam('Haidar', 'pagi')
